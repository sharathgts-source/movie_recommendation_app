# 🎬 Movie Recommendation App

A simple and responsive Movie Recommendation App developed using Flutter.  
The application displays movie posters, ratings, and descriptions with an attractive user interface.

---

# 📱 Features

- Display movie recommendations
- View movie posters and ratings
- Movie details screen
- Responsive UI design
- Dark theme interface
- Smooth navigation between screens
- Beginner-friendly Flutter project
- Works without API integration

---

# 🛠️ Technologies Used

- Flutter
- Dart
- Material Design
